# Personal Website

A static personal portfolio for Shaan with dynamic GitHub stats, project cards, social links, and live status integrations.

## Live Site

GitHub Pages URL:
[https://newbbd.github.io/Personal-website/](https://newbbd.github.io/Personal-website/)

## Features

- GitHub profile stats and repository cards
- Total repo, follower, and star counters
- Discord status integration through Lanyard
- Animated page title and visual effects
- Responsive layout for desktop and mobile
- Clean social link section

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub API
- Lanyard API
- GitHub Pages

## Project Structure

- `index.html`: Main page structure
- `styles.css`: Visual styling and responsive layout
- `script.js`: GitHub stats, Discord status, animations, and interactions
- `pfp.jpg`: Profile image and favicon

## Why I Built This

I built this as my personal web presence and project hub. The goal is to make my work easier to find, show what I am building, and keep the site fast without using a heavy framework.
